#作甚用的
------
质心上有位移的动作，如带位移的走路动作(不是原地走的那种)在MAX做走路时就把走动的距离都做出来了，那你就可以用这个工具把这个位移转到root 上的，这样导出的动画就可以在引擎中设置是否启用这个位移。

原地走路跑步动作都是要在引擎中由配动作的程序去大概给个距离，这种走路基本上都是脚掌打滑的。
